\# AI Cloud Berlin Hotel ESG Intelligence Choice Guide



Cloud-based AI dashboard for Berlin hotel ESG intelligence, sustainability analytics, and trusted hotel recommendations.



\## Features



\- ESG hotel scoring

\- Carbon hotspot maps

\- AI trust risk analysis

\- Berlin zone analysis

\- Top hotels by ESG performance

\- Sustainability recommendation engine

\- Interactive geospatial dashboard



\## Stack



\- Streamlit

\- FastAPI

\- Neon PostgreSQL

\- Render

\- Plotly

\- Python



\## Architecture



Neon PostgreSQL → Render FastAPI → Streamlit Cloud



\## Backend API



The dashboard connects to a live FastAPI backend hosted on Render and powered by a cloud PostgreSQL database on Neon.



\## Deployment



Deployed on Streamlit Community Cloud.
